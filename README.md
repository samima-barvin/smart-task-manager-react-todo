# Smart Task Manager – React Todo App

A simple and modern Todo application built using React.js.  
This app helps users manage daily tasks efficiently with features like add, edit, delete, and mark as completed. It also stores data in localStorage so tasks are saved even after refreshing the page.

---

## 🚀 Features

- ➕ Add new tasks
- ✏️ Edit existing tasks
- 🗑️ Delete tasks
- ✅ Mark tasks as completed
- 💾 Data saved in localStorage
- 📊 Task counter (Total & Completed)

---

## 🛠️ Tech Stack

- React.js (useState, useEffect)
- JavaScript (ES6+)
- HTML5
- CSS3
